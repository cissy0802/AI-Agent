Search Query Redirector

This JavaScript extracts search queries from URLs and redirects to alternative search engines.

Features

  • Extracts search query from current URL (supports Google, Bing, DuckDuckGo, Yahoo, Baidu, Yandex)
  • Constructs new URL for alternative search engine
  • Works as a bookmarklet or standalone script

Usage Examples

1. Basic Usage - Extract Query

// Get the search query from current URL const query = getSearchQuery(); console.log('Search query:', query);

2. Redirect to Alternative Search Engine

// Redirect to DuckDuckGo with the same query redirectToAlternativeSearch('duckduckgo'); // Redirect to Bing with the same query redirectToAlternativeSearch('bing');

3. Construct URL Without Redirecting

const query = getSearchQuery(); const bingUrl = constructSearchUrl(query, 'bing'); console.log('Bing URL:', bingUrl);

4. Get Available Alternative Engines

const alternatives = getAlternativeEngines(); console.log('Available alternatives:', alternatives);

Test the Functions

Open your browser console and try these functions. First, navigate to a Google search results page, then:

  1. Open Developer Tools (F12)
  2. Go to the Console tab
  3. Load the script: <script src="search_query_redirect.js"></script>
  4. Try: getSearchQuery() to see the current query
  5. Try: redirectToAlternativeSearch('duckduckgo') to redirect

Supported Search Engines

  • Google: Parameter: q
  • Bing: Parameter: q
  • DuckDuckGo: Parameter: q
  • Yahoo: Parameter: p
  • Baidu: Parameter: wd
  • Yandex: Parameter: text
